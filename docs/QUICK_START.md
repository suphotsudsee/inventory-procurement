# 🚀 Quick Start Guide

> 5 นาทีแรกกับระบบ Inventory & Procurement Management System

---

## 📋 สิ่งที่ต้องเตรียม

ก่อนเริ่มต้น ตรวจสอบว่าคุณมี:

- ✅ Node.js 18.x ขึ้นไป
- ✅ MySQL 8.0 ขึ้นไป
- ✅ เบราว์เซอร์ (Chrome, Firefox, Edge)

---

## ⚡ ติดตั้งและรันใน 5 นาที

### นาทีที่ 1: เตรียม Database

```bash
# เปิด Terminal และรันคำสั่งนี้
mysql -u root -p

# ใน MySQL shell:
CREATE DATABASE inventory_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
EXIT;
```

### นาทีที่ 2-3: ติดตั้ง Backend

```bash
# เข้าโฟลเดอร์ backend
cd backend

# ติดตั้ง dependencies
npm install

# สร้างไฟล์ .env
cat > .env << EOF
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=inventory_db
PORT=3001
NODE_ENV=development
JWT_SECRET=your-secret-key-change-in-production
EOF

# รัน database schema
mysql -u root -p inventory_db < database/schema.sql

# เริ่ม server
npm run dev
```

✅ **Backend พร้อมใช้งานที่** `http://localhost:3001`

### นาทีที่ 4-5: ติดตั้ง Frontend

เปิด Terminal ใหม่:

```bash
# เข้าโฟลเดอร์ frontend
cd frontend

# ติดตั้ง dependencies
npm install

# สร้างไฟล์ .env
echo "VITE_API_URL=http://localhost:3001" > .env

# เริ่ม development server
npm run dev
```

✅ **Frontend พร้อมใช้งานที่** `http://localhost:5173`

## 🆘 เคล็ดลับและความช่วยเหลือ

### เคล็ดลับการใช้งาน

1. **ใช้ FEFO อัตโนมัติ**: ระบบตัดสต็อกจาก Lot ใกล้หมดอายุก่อนเสมอ

2. **ตั้งค่า Min Stock**: กำหนดระดับสต็อกขั้นต่ำเพื่อรับการแจ้งเตือน

3. **ใส่ Expiry Date เสมอ**: ช่วยให้ระบบจัดการ FEFO ได้ถูกต้อง

4. **บันทึก Notes**: ใส่หมายเหตุทุกครั้งที่ปรับสต็อก

### ปัญหาที่พบบ่อย

| ปัญหา | วิธีแก้ |
|-------|--------|
| ล็อกอินไม่ได้ | ตรวจสอบ username/password หรือลองรีเซ็ต |
| ข้อมูลไม่แสดง | กดปุ่ม "รีเฟรช" หรือตรวจสอบ Database |
| ตัดสต็อกไม่สำเร็จ | ตรวจสอบว่ามีสต็อกเพียงพอหรือไม่ |
| PO ไม่ยอมบันทึก | ตรวจสอบว่ากรอกข้อมูลครบทุกช่อง |
